# GraphQL with Rails

This is a a basic Rails project where I implemented GraphQL queries and mutations. 

## Prerequisits
  * Ruby 2.6.3
  * Rails 5.2.8
  * GraphQL 2.0.14

## Install Dependencies
  * bundle

## Create and Migrate Database
  * Run the following command to create and migrate the database
  * rails db:create
  * rails db:migrate
